fimmta verkefnið mitt

1. Hvað gera eftirfarandi Linux skipanir?
	cd: breytit um skrá.
	cd ~: Tekur þig beint í heima skránna.
	ls: sínir þér hinnihaldið í skránna sem þú ert í.
	pwd: Skrifar út í hvaða ská þú ert í.
	mkdir: Býr til skrá ef hún er eki til.

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone: Clonar geymslu í nýa skrá.
	git log: Sýnir allar breytingar í geymslu í öfugarri röð.
	git status: Sýnir þér hvaða skrár git er að tracka og hvaða skrá hann er ekki að tracka.
	git diff: Sýnir þer munin á skýrsluni þinni fyrir og eftir breytingar.
	git checkout: Endurheimtar fyrverandi hinnihald skránna þinnar. 

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff: þessi skupun leifir þér að sjá allan mismun verkefnsins í gegnum tíðina.
	b) git diff --staged: þessi skupun leifir þér að sjá allan mismun staged sránna í gegnum tíðina.
	c) git diff commit1 commit2: þessi skipun leifir manni að sjámismunin á milli valdna commits.
						    
4. 	Hvað er útgáfustýring (e.version control)?: 

    Útgáfustýring geymir allar gerðir af verkefninu 
    þínu þannig þú getur alltaf skoðað hvernig verkefnið þitt var áður og ef þú þarft þá getur
	þú alltaf farið til baka og haldið áfram með gamla gerð af verkefninu þínu.
